### 1. 分层渲染系统
```
Layer 1: Background     → 背景渐变层
Layer 2: Floor         → 地板纹理层
Layer 3: Decoration    → 灯具装饰层  
Layer 4: Front Desk    → 前台服务层
Layer 5: Tables        → 餐桌摆放层
```

## 🚦 使用简易教程

### Step 1: 初始化系统 (30秒)
```js
const SceneLoader = require('./utils/scene-loader');

// 初始化
SceneLoader.init('myCanvas');  
await SceneLoader.prelodAllScenes();
await SceneLoader.renderDefaultScene();
```

### Step 2: 切换主题 (10秒)
```js
// 方法1
await SceneLoader.switchToTheme('烧烤店');

// 方法2 - 用UI
this.setData({ showThemeSelector: true });
```

### Step 3: 配置自定义 (可选)
```json
// config/scene-layout.json 
{
  "totalTable": 10,      // 改桌数
  "occupiedRatio": 0.8,  // 改占用率  
}
```
